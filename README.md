# לוח הטהרה החכם

לוח הטהרה החכם להצגת עונות פרישה.

מאגר זה משמש להפצת **Releases רשמיים בלבד** (Windows EXE + Android APK).
**קוד המקור אינו זמין במאגר זה.**

## הורדת הגרסה האחרונה

לכל גרסה רשמית מצורפים שני קבצי התקנה:

- **Windows:** `Smart-Tahara-Calendar-Setup-vX.Y.Z.exe`
- **Android:** `Smart-Tahara-Calendar-vX.Y.Z.apk`

ניתן להוריד את הגרסה העדכנית ביותר מהקישור הבא:

➡ [**Releases**](../../releases/latest)

## בדיקת עדכונים מתוך התוכנה

האפליקציה כוללת מנגנון בדיקת עדכון אוטומטי בלשונית **"מערכת"** בהגדרות.
הבדיקה מתבצעת מול עמוד ה-Releases של מאגר זה ומציעה הורדה של גרסת ההתקנה
המתאימה לפלטפורמה הנוכחית (Windows / Android).
