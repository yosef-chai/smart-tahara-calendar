# לוח הטהרה החכם

לוח הטהרה החכם להצגת עונות פרישה.

מאגר זה משמש להפצת **מהדורות רשמיות** (Windows EXE + Android APK) של מערכת לוח הטהרה החכם.

## הורדת הגרסה האחרונה

לכל גרסה רשמית מצורפים שני קבצי התקנה:

- **Windows:** `Smart-Tahara-Calendar-Setup-vX.Y.Z.exe`
- **Android:** `Smart-Tahara-Calendar-vX.Y.Z.apk`

ניתן להוריד את הגרסה העדכנית ביותר [**כאן**](../../releases/latest).
